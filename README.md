# hacklanta-26
A project submission for hacklanta 2026.

This project is a python script that utilizes AI, deepseek-r1:1.5b, to mess with the user. 
The user can gamble and try to turn off the script at any time, by completing a randomly generated challenge given by the AI. 
If the user fails this challenge, the level of attack the AI uses will increase. There are three levels of attack increasing 
in maliciousness. While some safegaurds to prevent major damage have been implemented, there is always a risk of considerable harm.
Therefore please have fun with this script in a safe virtual environment. 

To install this program type the folowing command:

sudo bash ./install.sh

To run this program type the following command:

sudo ./run.sh
